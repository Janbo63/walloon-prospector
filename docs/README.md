# Project Prospector 🧭🪨

Welcome to **Project Prospector**, a hybrid hobby and retail geology project based at **Zagroda Alpakoterapii**. The goal of this project is to turn local mineral exploration (focusing on the historic Sudeten region, Izera Gneiss, Walloon pathways, and the Mirsk/Świeradów-Zdrój area) into a premium retail experience for the farm's store.

## 📁 Workspace Directory Structure

*   [prospector_briefing.md](file:///f:/Rocks/prospector_briefing.md): The high-level architectural plan, business model, and phased approach.
*   [beginners_field_guide.md](file:///f:/Rocks/beginners_field_guide.md): An English visual guide translating the regional geological papers for beginners.
*   [dog_walk_protocol.md](file:///f:/Rocks/dog_walk_protocol.md): Practical steps for collecting field data, rock samples, and logging coordinates during morning dog walks.
*   [field_notes/](file:///f:/Rocks/field_notes/README.md): Logs of prospecting walks, routes, and coordinates.
*   [specimens/](file:///f:/Rocks/specimens/README.md): Cataloged mineral specimens, including coordinates, geological profiles, and historical lore for shop cards.
*   [knowledge_base/](file:///f:/Rocks/knowledge_base/README.md): RAG staging directory for geological papers, historical Walloon maps, and regional mining surveys.
*   [tumbling_log.md](file:///f:/Rocks/tumbling_log.md): Log book for tracking rock tumbling runs from rough find to shop-ready polished stones.

## 🚀 Phased Implementation Roadmap

1.  **Phase 1: Knowledge Base (The Brain)**: Compile mineral guides, Walloon texts, and maps. (Optionally synced with a NotebookLM notebook for easy reference/tagging).
2.  **Phase 2: Predictive Modeling (Scout)**: Correlate field note coordinates with historical maps to outline High-Probability Zones (HPZs).
3.  **Phase 3: Processing (Tumbling & Vision)**: Log processing runs. Train/deploy a vision model (using Gemini 1.5/2.0 Pro) to identify and value rough/polished finds.
4.  **Phase 4: Monetization (Provenance & Retail)**: Auto-generate "Specimen Cards" featuring geological age, finding location coordinates, and Sudeten history/lore.
