# Field Notes Directory 📝⛰️

Use this directory to store your daily or weekly prospecting logs. Each walk should be recorded in a new markdown file named in the format `YYYY-MM-DD-walk-name.md`.

---

## 📄 Walk Log Template

Copy and paste the template below to document your walks:

```markdown
# Prospecting Log: [Walk Name / Route]

*   **Date**: YYYY-MM-DD
*   **Time**: Start Time - End Time
*   **Weather**: Sunny / Muddy / Raining
*   **Dog Companion**: [Dog Name(s)]
*   **Route Map / GPX File**: [Link or attachment]

## 🗺️ Route Overview
Brief description of where you went (e.g., "Walked the stream path heading north towards Mirsk border, explored the uprooted spruce tree area").
```

## 📂 Mapping & Visualizing
If you collect GPX files, save them in this folder as `YYYY-MM-DD-walk-name.gpx`. These can be read by GIS tools or eventually parsed by the "Scout" module to generate your proprietary exploration heat map.
